# K1 新服大地图重构策划案

**版本**:v0.6 精简稿(移除工作量估计)  
**日期**:2026-07-02  
**适用范围**:仅新开服务器,不做老服兼容  
**性质**:借用 KVK 地图骨架与占领规则,重构新服大地图形态。**逻辑上仍是原服**,不是 KVK。

参考依据:K1 KVK/活动/新玩家相关口径见 `wiki/work/k1-kvk-burning-expedition.md`、`wiki/work/k1-live-activity-patterns-2026h1.md`、`wiki/work/k1-new-player-retention-roi.md`;兼容风险见 [[高兼容性风险修改项自检]]。落地后的付费成长效果验证见 [[k1-new-map-payment-growth-attribution]]。

---

## 一、方案概述

### 1.1 一句话说明

新服大地图放弃现有"平原自由摆方尖碑"形态,改用 **KVK 地图骨架(方格化 + 堡垒归属 + 4 扇形出生区 + 中心王座)** 作为地形结构,配合"占领相邻堡垒扩张领地"的核心占领规则;原服商业化主结构和 KVK 参战规则保持不变,但地图相关活动、任务、成就、排行计数需要重新定义。

### 1.2 核心变化

| 维度 | 现有原服 | 新服方案 |
|---|---|---|
| 地图结构 | 7200x7200 平原,自由落城 | 7200x7200,方格化划分,4 角扇形出生区,中心王座区 |
| 领地扩张 | 建方尖碑连接,建中心/远征堡垒 | 占领相邻堡垒方格 |
| 联盟领地判定 | 方尖碑覆盖区域 | 已占领堡垒方格集合 |
| 龙巢 | 散布在地图任意位置 | 位于 4 个相邻堡垒交叉区域,独立于堡垒方格 |
| 圣坛 | 独立点位 | 嵌入到特定堡垒方格内 |
| 王座 | 无 | 中心区可占领,战斗与重置复用现有王座战规则 |
| 高级区域访问 | 无限制 | 服务器共享迷雾,逐步开放 |
| 出生区域 | 9 区划分 | 4 扇形区(仅出生分流,无阵营含义) |
| KVK 参战 | 现有规则 | 报名、进入、回归规则不变 |
| 活动/任务/排行 | 基于原地图目标 | 地图相关计数重配为堡垒、龙巢、都城/王座等新目标 |
| 商业化 | 现有规则 | 主结构不变,仅校验地图相关付费道具/权益是否受影响 |

### 1.3 目标

- 新服玩家从落地起就在有战略结构的地图上,不再是松散平原。
- 地图有天然地标和争夺焦点(4 角扇形、中心王座、堡垒方格、交叉龙巢),提升联盟对抗的战略感。
- 移除自行修建方尖碑/联盟堡垒来连接龙巢的理解门槛,改为联盟直接围绕系统堡垒和龙巢组织行动。
- KVK 参战时玩家已经熟悉方格/堡垒机制,降低活动学习成本。
- 通过迷雾进度化开放,形成新服自然的赛季感。

### 1.4 非目标

- 不重构 KVK 玩法本身。
- 不改造老服。
- 不改动联盟系统的基础规则(报名、退出、弹劾、机器人、盟主竞选、联盟科技主干等)。
- 不重构商业化模型。
- 不引入 KVK 阵营、跨服匹配、天气、女神等完整 KVK 活动机制到原服。

### 1.5 设计模式旁注

本方案采用"结构相似、规则不完全对齐"的思路:新服原服地图借用 KVK 的方格、堡垒、王座骨架,但保留原服逻辑和商业化主结构。这与 [[双模式难度故意不对齐]] 的原则一致:两个模式共享认知锚点,但允许规则偏向不同,让玩家自然迁移而不是要求完全等价。

## 二、地图结构

### 2.1 地图基本参数

- **尺寸**:7200x7200(与现有大地图/KVK 地图一致)
- **方格大小**:300x300(复用 KVK)
- **总方格数**:576 格(24x24),需扣除中心王座区、边界不可用区,实际可用堡垒方格约 400-500 个(具体数量按配表)
- **区域构成**:堡垒方格 + 龙巢交叉点 + 王座方格 + 圣坛附属点

### 2.2 分区划分

| 分区 | 位置 | 说明 |
|---|---|---|
| 扇形出生区 | 4 个角落 | 玩家出生落城,PVE 强度最低,含初始共享堡垒 |
| 中间圈 | 扇形区与中心之间 | 第一层迷雾覆盖,中级 PVE,含部分龙巢/圣坛 |
| 中心圈 | 地图中心区域 | 第二层迷雾覆盖,高级 PVE,含王座与高级龙巢/圣坛 |
| 王座方格 | 地图正中心 | 独立配置,复用 K1 王座战规则 |

### 2.3 堡垒方格属性

每个堡垒方格包含以下字段:

- **方格 ID**:唯一标识
- **归属联盟 ID**:0 表示中立/共享
- **堡垒类型**:初始共享堡垒 / 普通堡垒 / 王座关联堡垒(如需要)
- **附属圣坛**:0 或 1 个 + 圣坛类型(智慧泉/远古矿坑/金矿场/英灵殿/工具屋 之一)
- **相邻龙巢 ID 列表**:0-N 个,由 4 格交叉点决定
- **PVE 强度档位**:1-6 级,按分区配置
- **迷雾层级**:0(常开)/ 1(第一层迷雾)/ 2(第二层迷雾)
- **相邻方格 ID 列表**:4 或 6 邻居(方格拓扑决定)
- **免战截止时间**:堡垒属性,配置可调,扩展性更好。当前仅由联盟首战写入,未来可扩展至其他机制。

### 2.4 初始共享堡垒

- **性质**:永久中立,不可被任何联盟占领。
- **作用**:所有联盟扩张的公共起点,新联盟可以此为锚点向外攻占。
- **数量**:每个扇形出生区配置 4-8 个(具体按预期同扇形联盟数)。
- **分布**:均匀散布在扇形区内,保证任何出生点玩家都能找到相邻初始堡垒。
- **相邻规则**:初始堡垒相邻的普通堡垒属于公共首战圈。该属性长期有效。玩家侧不展示"冷启动"概念;没有任何已占领堡垒的联盟,只能攻占初始共享堡垒相邻的非免战普通堡垒,目标可为中立堡垒,也可为已被其他联盟占领但免战已结束的堡垒。占领成功后触发免战时间(见 4.5)。

### 2.5 王座方格

- **位置**:地图正中心。
- **迷雾层级**:2(需两层迷雾都开放才可进入)。
- **占领与重置规则**:**完全复用 K1 现有王座战/跨服王座战规则**,不重新设计战斗、占领、重置与奖励逻辑。
- **归属方**:某个联盟。
- **收益**:按现有王座战配置提供联盟积分与战斗 buff。若新地图入口或开放时间需要差异,仅新增适配配置,不改王座战核心规则。

## 三、出生分配

### 3.1 出生区数量

- 从现有 9 区改为 **4 个扇形出生区**。
- 4 个扇形仅作**出生分流**,同服玩家均为友军,无阵营对抗含义。

### 3.2 出生分配算法

- **开服首 X 天**(建议 7 天,可配):新玩家自由选择落城点,4 个扇形区任选。
- **X 天后**:新玩家自动分配到 4 个扇形区中人数最少的一个,避免头部联盟集中。
- **联盟归属不限扇形**:联盟成员可分布在任意扇形区,联盟迁城可跨扇形。

### 3.3 新玩家保护

- 新玩家出生后享受 **新手保护罩**(时长复用原服规则)。
- 若新玩家落入已被联盟占领的方格,保护罩内不可被攻击。
- 保护罩失效前迁移到无归属或己方联盟方格,不受影响。

### 3.4 扇形区容量

- 单服玩家全部落在 4 个扇形区,人口密度约为 KVK 时单扇形装 1 服的水平。**新服不需要扩大扇形面积**,现有 KVK 扇形尺寸可直接复用。
- 若某扇形区落城密度接近上限,新玩家分配到其他扇形。

## 四、堡垒占领规则

### 4.1 相邻扩张

**核心规则**:一个联盟可攻击的堡垒 = 已占领堡垒的相邻方格 + 初始共享堡垒的相邻方格。

- 联盟成立时无任何堡垒,首个可攻击目标 = 任一初始共享堡垒相邻的普通堡垒。
- 占领后,该堡垒相邻的所有堡垒进入"可攻击列表"。
- 扩张连锁,无跳跃。

### 4.2 攻击资格

- 只有满足"相邻"条件的联盟可发起攻击。
- 攻击方需从联盟成员中派兵(集结或单人攻击,规则复用现有联盟建筑攻击)。
- 攻击成功后堡垒归属变更,原占领方失去该方格。
- 若目标堡垒处于免战时间内,不可发起攻击。

### 4.3 归属规则

- **堡垒归联盟不归个人**:占领贡献者退盟不影响归属。
- **联盟解散**:所属堡垒 24 小时内自动变为中立,可被任何相邻联盟重新占领。
- **归属衰减**:堡垒无归属衰减,占领后永久归属,除非被打下或联盟解散;龙巢有失效状态,详见 §5.1。
- **王座例外**:王座方格按现有王座战规则重置。

### 4.4 联盟领地判定

- **联盟领地** = 该联盟已占领的堡垒方格集合。
- **领地保护科技**触发条件:玩家城堡所在方格属于自己联盟(现有科技保留,只是"联盟领地"定义变了)。
- **联盟迁城落点**:必须在联盟领地内,且在盟主城堡附近(距离限制沿用现有规则)。

### 4.5 新联盟首战规则

- **玩家视角**:没有"冷启动"概念。联盟若当前没有任何已占领堡垒,地图只显示初始共享堡垒相邻的非免战普通堡垒为可攻打目标。
- **系统判定**:联盟占领堡垒数 = 0 时,可攻击任一初始共享堡垒相邻第一圈的非免战普通堡垒。
- **目标范围**:
  - 可以是中立堡垒。
  - 可以是已被其他联盟占领、但免战已经结束的堡垒。
  - 不可以是免战中的堡垒。**免战优先级高于首战攻击资格**。
- **首战成功**:
  - 占领任意 1 个堡垒后,该联盟后续按普通相邻扩张规则行动。
  - 首次占领成功后,该堡垒进入免战时间,持续 T 小时(建议 12-24 小时,具体可配)。
- **免战限制**:
  - 免战期间该堡垒不可被其他联盟攻击。
  - 免战不保护该联盟的其他堡垒。
  - 若新联盟主动从免战堡垒继续发起攻击,是否提前解除免战待定。
- **防滥用**:同一玩家在 30 天内解散并重建联盟,是否限制其再次触发首战免战由配置决定;系统日志记录联盟解散、创建、盟主变更和关键成员迁移事件用于判定。

### 4.6 KVK 期间的堡垒归属

- 玩家参战 KVK 期间,其原服堡垒/联盟领地归属**保持不变**,数据完整保留。
- 玩家可随时从 KVK 战场返回原服(复用现有规则)。
- 原服堡垒在 KVK 期间不冻结,仍可被其他联盟攻击。
- KVK 开启期内,原服**普通堡垒占领所需时间统一按固定倍率延长**(全服生效,不区分联盟参战比例)。倍率由配表控制,建议初始值 2-3 倍。
- 王座方格不适用占领时间延长,以"王座战完全复用现有规则"为优先约束。

## 五、龙巢与圣坛

### 5.1 龙巢

- **分布**:龙巢位于 4 个相邻堡垒的交叉区域,不位于任何单个堡垒方格内部。
- **地图表达**:龙巢作为独立战略点显示在四格交界处,其攻击入口由相邻堡垒提供。
- **攻击资格**:占领任一相邻堡垒的联盟,可对该龙巢发起攻击。
- **占领规则**:
  - 龙巢占领是独立战斗行为,不随堡垒自动归属。
  - 龙巢归属独立于相邻堡垒归属。
  - 同一龙巢最多有 4 个方向的相邻堡垒可提供攻击资格。
- **归属解释**:龙巢不再被描述为"某个堡垒里的附属物",而是"多个堡垒之间的争夺点",以降低玩家对归属的困惑。
- **失去相邻堡垒后的处理**:归属联盟失去全部相邻堡垒时,龙巢进入**失效状态**(复用现有龙巢已有的失效状态,不新增机制)。失效状态下保留原归属但加成暂停;任一联盟占有相邻堡垒即可发起争夺;原归属联盟重新获得任一相邻堡垒且龙巢未被其他联盟打下时,龙巢恢复生效。
- **数量上限**:由联盟科技新分支决定(初始 X,满级 8,复用现有 8 个上限)。
- **加成**:复用现有龙巢加成规则(联盟建筑战力)。
- **分布位置**:主要分布在中间圈和中心圈,扇形出生区不放龙巢。

### 5.2 圣坛

- **类型**:5 种(智慧泉/远古矿坑/金矿场/英灵殿/工具屋),加成规则完全复用现有。
- **分布**:嵌入到特定堡垒方格内,位置固定,由配表决定。
- **占领规则**:**占领方格 = 自动获得圣坛加成**。与龙巢不同,圣坛不作为独立占领目标。
- **数量**:按现有 5 种圣坛各 N 个的配比分布,建议中间圈和中心圈各放一组。
- **加成生效**:占领方格的联盟全员享受该圣坛加成。

## 六、迷雾系统

### 6.1 迷雾规则

- **服务器共享**:全服迷雾状态同步,不是玩家个人视野。
- **不可付费解锁**:任何个人付费不能提前解锁迷雾。
- **复用现有迷雾规则**:项目已有迷雾系统,只需新配区域和开放时间。

### 6.2 迷雾层级与开服进度

- **迷雾层级 0**:常开区域,含 4 个扇形出生区和初始共享堡垒。开服即可用。
- **迷雾层级 1**:中间圈(区域 2),**开服 D8 开放**。
- **迷雾层级 2**:中心圈,**开服 D15 开放**(区域 3;王座此时可见但未开启)。
- **首次本服王座开放**:**开服 D21**,王座可被攻击/占领,规则完全复用现有王座战。

新服进程整体对标 KVK 3 周节奏,但省掉 KVK 前 7 天的报名和匹配阶段,直接从 D0 开打,21 天完成"扇形→中间圈→中心圈→王座"完整进度。

### 6.3 迷雾状态下的表现

- 方格、堡垒 icon **可见**,但归属信息隐藏。
- 龙巢/圣坛 icon **可见**,类型和归属信息隐藏。
- 部队和城堡**不可进入**迷雾覆盖的方格。
- 迷雾开放后一切正常显示,城堡可迁入。

### 6.4 迷雾开放触发

- **纯时间驱动**(推荐):按开服日期到达,自动全服解锁。
- 备选:混合驱动(时间 + 全服进度达成),若纯时间在灰度中不理想再切换。

### 6.5 KVK 期间的迷雾

- 迷雾开放时钟基于服务器创建时间累计,KVK 期间**不暂停**。
- 玩家在 KVK 战场期间原服迷雾仍按时开放,回来时可能已有新区域可探索。

## 七、PVE 强度分布

### 7.1 强度梯度

| 分区 | 野怪等级 | 火龙巢等级 | 恶魔守卫 |
|---|---|---|---|
| 扇形出生区 | 1-3 级 | 1-3 级 | 无 |
| 中间圈 | 3-5 级 | 3-5 级 | 部分坐标刷新 |
| 中心圈 | 5-6 级 | 5-6 级 | 大部分坐标刷新 |
| 王座方格 | 特殊配置 | 无 | 王座守卫(复用王座战规则) |

### 7.2 野怪与泰坦

- 分布规则复用现有原服野怪/泰坦刷新逻辑。
- 等级按 7.1 强度分布配置。
- 集结/自动集结/宝箱奖励等规则完全不变。

### 7.3 火龙巢

- 等级下调(从 KVK 的 6-9 级下调到 1-6 级梯度)。
- 归属规则与龙巢一致:位于多个堡垒交叉区域,占领任一相邻堡垒的联盟可发起攻击。
- 分布:各分区均有,数量按配表。

### 7.4 恶魔守卫

- 定位:与 KVK 现有玩法中的**中立 NPC**同一类,原服改为 KVK 地图结构后,该 NPC 同样刷新在原服。
- 与泰坦/野怪同层级的野生 PVE,所有玩家均可攻击。
- 分布:主要在中间圈和中心圈,由地图配表决定坐标。
- 刷新:按周期刷新,击败后经配置周期(建议初始 7-14 天,可配)重新在原坐标出现,复用现有原服中立 NPC 刷新逻辑。

### 7.5 龙巢守卫

- 定位:龙巢内部数据,**不显示在地图**,属于攻击龙巢的固有战斗流程。
- 攻击龙巢的联盟需先击败龙巢守卫才能完成占领,规则复用现有龙巢战斗流程。
- 不属于地图中立 NPC,不影响堡垒占领资格。

## 十、KVK 兼容

**核心原则**:KVK 玩法完全不变,新服地图作为"新的原服形态"承担 KVK 报名/回归两端。

### 10.1 KVK 报名

- 玩家/联盟报名 KVK,规则完全不变。
- 报名后玩家可选择进入 KVK 战场。

### 10.2 KVK 期间

- 玩家在 KVK 战场作战,规则完全不变。
- 玩家可**随时返回原服**(复用现有规则,无需开发)。
- 玩家在原服的堡垒/联盟领地归属**数据完整保留**。
- 玩家不在原服期间,其城堡的战斗状态遵循现有原服规则。
- 原服普通堡垒占领所需时间按固定倍率延长,详见 §4.6。

### 10.3 KVK 结束

- 玩家回归原服,规则完全不变。
- 原服地图上的联盟领地/堡垒归属/龙巢/圣坛/王座归属**不变**(除非在此期间被打下)。

### 10.4 迷雾与 KVK

- 迷雾开放时钟基于服务器创建时间累计,KVK 期间不暂停。
- 玩家 KVK 归来后可能发现原服有新迷雾开放,视为正常的服务器进度。

## 十一、活动、任务、排行、商业化

### 11.1 总原则

商业化主结构、原服常规活动框架、PVE/PVP 基础战斗规则尽量保留;但所有依赖旧地图结构的活动、任务、成就、排行榜计数必须重新定义。尤其是原方尖碑、中心堡垒、远征堡垒、龙巢连接相关内容,需要替换为新地图目标。

### 11.2 需要重定义的计数

| 旧目标/旧计数 | 新目标/新计数 |
|---|---|
| 建造方尖碑 | 占领普通堡垒 |
| 修建/升级联盟中心堡垒 | 占领堡垒总数 / 联盟领地方格数 |
| 修建/升级远征堡垒 | 占领中间圈/中心圈堡垒 |
| 连接龙巢 | 占领龙巢相邻堡垒 / 攻击并占领龙巢 |
| 占领龙巢 | 占领交叉点龙巢 |
| 占领圣坛 | 占领带圣坛的堡垒方格 |
| 王座相关目标 | 占领都城/王座、参与王座战、获得王座积分 |
| 联盟领地面积 | 联盟占领堡垒方格数 |
| 联盟建筑战斗 | 堡垒战 / 龙巢战 / 王座战 |

### 11.3 任务与成就

- **主线任务**:删除或替换"建造方尖碑""连接龙巢""建造联盟堡垒"类目标。
- **联盟任务**:新增"首次占领堡垒""占领 N 个堡垒""参与堡垒战""占领龙巢""占领带圣坛堡垒""进入中间圈/中心圈"等目标。
- **成就**:新增或替换联盟扩张类成就,计数按堡垒、龙巢、都城/王座、圣坛、领地方格数计算。
- **引导**:新手联盟引导必须解释"公共初始堡垒 -> 相邻堡垒 -> 龙巢交叉点 -> 中心圈/王座"的路径。
- **兼容**:任务/成就类型和数值属于高兼容风险项,旧版本客户端和旧服任务链需做隔离或降级。

### 11.4 活动与排行

- **最强领主/联盟排行类活动**:若原计数包含联盟建筑、方尖碑、龙巢连接,需映射为堡垒/龙巢/王座相关计数。
- **王座战**:战斗、重置、奖励、buff 复用现有规则;只适配地图入口、开放时间和都城/王座位置。
- **联盟活动**:涉及联盟领地、联盟建设、联盟建筑战斗的任务需重配。
- **PVE 活动**:野怪、泰坦、采集等基础计数保持不变;如果活动要求特定区域进入或怪物等级,需结合迷雾层级调整。
- **排行口径**:新增或替换联盟占领堡垒数、龙巢数、都城/王座积分、领地方格数等排行指标。

### 11.5 商业化

- 商业化模型不重构。
- 迁城、保护罩、加速、赛季卡、联盟相关权益需校验是否受新领地定义影响。
- 若某付费权益与方尖碑/联盟建筑/龙巢连接直接绑定,需要改描述或替换生效条件。

## 十二、风险与关注点

### 12.1 王座 buff 长期集中

- **风险**:2 周重置周期内占领联盟持续享受 buff,可能挤压其他联盟。
- **缓解**:坚持复用现有王座战规则,不额外加强 buff;通过中心圈方格数量和公共通路降低头部联盟完全封锁概率。

### 12.2 迷雾开放的进度感

- **风险**:纯时间驱动可能让新服玩家在 D8 前感觉"地图太小",D21 后感觉"扩张停止"。
- **缓解**:D8/D15/D21 是初始配置(对标 KVK 21 天节奏),灰度期间根据玩家节奏调整;必要时切换为混合驱动。

### 12.3 新联盟首战规则滥用

- **风险**:玩家可能解散老联盟、重建新联盟蹭首战免战,或用小号联盟骚扰头部联盟。
- **缓解**:首战可攻击目标仅限初始共享堡垒相邻第一圈的非免战普通堡垒;占领后进入普通相邻扩张;是否限制同一玩家 30 天内重复触发首战免战由配置决定;记录联盟解散、重建、盟主和关键成员迁移日志。

### 12.4 首战攻击既有堡垒的公平性

- **风险**:既有联盟会认为已占领堡垒被零堡垒联盟攻击不公平。
- **缓解**:可攻击范围必须清晰可视;只允许初始共享堡垒第一圈且必须非免战;被攻击方可获得战报提示和反攻倒计时;首占免战时间不能过长。

### 12.5 UI 信息密度

- **风险**:每个方格要显示归属、龙巢、圣坛、迷雾、PVE、免战等多层信息。
- **缓解**:客户端分层显示。缩略视图只显示归属颜色和关键图标,点击方格或龙巢显示详情弹窗。

### 12.6 首个 KVK 周期的验证

- **风险**:新地图 + KVK 参战往返机制是否与现有 KVK 系统兼容,需实测。
- **缓解**:灰度期覆盖至少一个完整 KVK 周期,重点验证原服堡垒占领时间延长、回归后迷雾开放和堡垒归属保留。

### 12.7 头部联盟提前占据王座周边

- **风险**:即便王座按现有规则重置,若中心圈其他方格被头部联盟占满,其他联盟难以接近王座。
- **缓解**:中心圈方格数量足够多(建议至少 30-40 格),避免被单一联盟垄断;必要时在中心圈引入更多中立堡垒作为公共扩张锚点。

### 12.8 活动/任务/排行不可完成

- **风险**:旧任务仍要求方尖碑、中心堡垒、远征堡垒或旧龙巢连接,导致新服模板下无法完成。
- **缓解**:上线前输出一张"旧计数 -> 新计数"替换表,覆盖主线、支线、联盟任务、联盟成就、活动排行和客服 FAQ。

### 12.9 新旧版本兼容

- **风险**:大地图 battle、跨场景逻辑、新大地图对象、任务/成就类型、实体新增字段、多语言参数都属于高兼容风险。
- **缓解**:新服模板与旧服模板隔离;旧客户端遇到未知地图对象、任务类型和多语言参数时必须降级;测试需拷线上数据回归。
